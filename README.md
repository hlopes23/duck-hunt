# Duck Hunt

A fun, interactive web-based version of the classic Duck Hunt game.

## Features

- Shoot moving ducks to score points
- Responsive design for desktop and mobile
- Sound effects and animations
- Score tracking


## Technologies Used

- HTML
- CSS
- JavaScript (ES6+)
